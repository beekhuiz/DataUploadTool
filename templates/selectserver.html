Select server

Select the storage server. There are three different server types:

Regular file server
- Suitable for all file types

Geoserver (not yet supported)
- can store various geodata formats, such as shapefiles and GeoTIFFs. Data can be accessed using WMS and WFS.

THREDDS Data Server (TDS)
- a web server that provides data access for scientific grid datasets (such as NetCDF), using a variety of web services
